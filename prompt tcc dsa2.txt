# PROMPT PARA CRIAÇÃO DO PROJETO TCC

## Nome do Projeto

**SGT – Sistema de Gerenciamento de Tarefas e Organização Acadêmica**

---

# TAREFA

Desenvolver um sistema web completo utilizando exclusivamente ReactJS com Vite, focado na organização acadêmica de estudantes do ensino médio, ensino técnico, vestibulandos, universitários e grupos de TCC.

O sistema deverá ser moderno, responsivo, intuitivo, acessível e preparado para hospedagem na plataforma Vercel.

A aplicação deve permitir que estudantes gerenciem suas matérias, tarefas, metas, prazos, notas e produtividade acadêmica em um único ambiente digital.

O projeto deverá seguir boas práticas de desenvolvimento Frontend, componentização, organização de código, responsividade, experiência do usuário (UX) e interface moderna (UI).

---

# CONTEXTO

O projeto será desenvolvido como Trabalho de Conclusão de Curso (TCC) por alunos do curso Técnico em Informática.

O curso possui disciplinas voltadas para:

* Desenvolvimento Web
* ReactJS com Vite
* Banco de Dados
* Engenharia de Software
* Modelagem de Sistemas
* Desenvolvimento Mobile
* Experiência do Usuário (UX)
* Gestão de Projetos

Após análise da realidade escolar, foi identificado que muitos estudantes apresentam dificuldades para organizar suas atividades acadêmicas, controlar prazos de entrega, acompanhar o próprio desempenho e manter uma rotina eficiente de estudos.

Grande parte das plataformas educacionais existentes concentra-se apenas em disponibilizar materiais, notas ou comunicação institucional, sem oferecer ferramentas completas para planejamento pessoal dos estudos.

Dessa forma, surge a necessidade de um sistema centralizado capaz de auxiliar o estudante em sua rotina acadêmica diária.

O projeto está alinhado ao Objetivo de Desenvolvimento Sustentável número 4 da ONU:

**ODS 4 — Educação de Qualidade**

O sistema deverá contribuir para:

* Organização dos estudos
* Planejamento acadêmico
* Desenvolvimento da autonomia estudantil
* Redução de atrasos em entregas
* Melhoria da produtividade
* Melhor gerenciamento do tempo
* Apoio à aprendizagem contínua

---

# PROBLEMA IDENTIFICADO

Atualmente muitos estudantes enfrentam dificuldades como:

* Esquecimento de trabalhos e atividades;
* Perda de prazos importantes;
* Acúmulo de tarefas;
* Falta de planejamento semanal;
* Dificuldade para acompanhar notas;
* Baixa produtividade;
* Falta de controle das disciplinas;
* Dificuldade em trabalhar em grupo;
* Ausência de uma plataforma única para gestão acadêmica.

Esses problemas geram:

* Estresse;
* Sobrecarga;
* Queda de desempenho escolar;
* Falta de organização;
* Dificuldades no gerenciamento do tempo;
* Redução da qualidade das entregas.

---

# SOLUÇÃO PROPOSTA

Criar uma plataforma web moderna chamada SGT que permita ao estudante controlar toda sua rotina acadêmica através de um ambiente único.

O sistema deverá fornecer ferramentas para:

* Organização de disciplinas;
* Gerenciamento de tarefas;
* Controle de prazos;
* Acompanhamento de notas;
* Visualização de produtividade;
* Planejamento semanal;
* Comunicação entre estudantes;
* Organização de grupos de TCC.
